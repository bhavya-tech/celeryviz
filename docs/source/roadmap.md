# Roadmap & Future Plans

We are constantly working to improve `celeryviz`. Here is what we have planned:

## Upcoming Features

- **Advanced Filtering**: Filter tasks by name, state, or timestamp in the UI.
- **Persistence Support**: Save task history to a database for long-term analysis.
- **Enhanced Security**: Optional authentication for the visualization server.

## Contributing

We welcome contributions! Check out our [GitHub issues](https://github.com/bhavya-tech/celeryviz/issues) to see what we are working on and how you can help.
